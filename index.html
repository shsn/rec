Recipe Site

Getting Started

Paprika's built-in web browser lets you search the internet for recipes to download. Our natural language processing engine can automatically download recipes written in English from most websites.

Search for Recipes

Note You can always return to this search page using the first bookmark in Paprika's bookmarks list.

How to Download a Recipe

  1. Use the browser to find a recipe you would like to download.
  1. Press the Download button in the lower right corner of the toolbar.
  1. Paprika will attempt to download the recipe from the page. Press the Save button to save it.
  1. If Paprika cannot figure out how to download the recipe automatically, you can use the clipboard tools (described in the User Guide) to clip the recipe manually.

Note The recipe needs to be loaded and visible in the browser before you press the Download button.
